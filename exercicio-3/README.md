# tinnova-desafio

## 1. Iniciando 

### 1.1 Requísitos 

- NodeJs (https://nodejs.org/)

### 1.2 Configurações do Projeto 

Comece clonando o repositório para sua máquina. 

```sh
git clone https://github.com/lpontes7/tinnova-desafio.git
```
### 1.3 Instalar dependências do projeto

Use o comando 

```sh
npm i 
```

### 1.4 Rodar Projeto

Use o comando 

```sh
node index.js
```
